# ImageLoaders
这是一个对比当前Android上主流图片加载库的项目，参与对比的项目有UIL,Picasso,Volley-ImageLoader,Fresco,Glide。

###加载速度测试
分别在从网络加载，从磁盘缓存加载，从内存缓存中加载测试20次。
